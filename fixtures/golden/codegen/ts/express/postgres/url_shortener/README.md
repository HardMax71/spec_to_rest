# UrlShortener

Generated TypeScript service (TypeScript + Express + Prisma + PostgreSQL).

## Quick start

```bash
cp .env.example .env
docker compose up --build
curl http://localhost:8080/health
```

## Local dev

```bash
npm install
npx prisma generate
npm run typecheck
DATABASE_URL='postgresql://url_shortener:url_shortener@localhost:5432/url_shortener?schema=public' npm run dev
```

## Database migrations

Migrations under `prisma/migrations/` are generated by `spec-to-rest compile` — do not
edit `prisma/schema.prisma` or run `prisma migrate dev` directly (it would conflict with
pre-generated migrations).

```bash
make migrate-up     # npx prisma migrate deploy
make migrate-down   # roll back the latest applied migration (uses down.sql + prisma migrate resolve)
```

To change the schema, edit the upstream `.spec` and re-run `spec-to-rest compile`; the
compiler emits a new `NNN_schema_update/` directory with both `migration.sql` and
`down.sql` derived from the spec diff.
