# UrlShortener

Generated by [spec-to-rest](https://github.com/HardMax71/spec_to_rest) from `UrlShortener.spec`. Do not edit files in
`app/`, `alembic/`, `openapi.yaml`, or the infrastructure files at the project root by hand —
they will be overwritten on the next regeneration.

## Quick start (Docker)

```bash
cp .env.example .env
make docker-up
curl http://localhost:8000/health
curl http://localhost:8000/ready
curl http://localhost:8000/metrics
```

Traces are opt-in: set `OTEL_EXPORTER_OTLP_ENDPOINT` (for example
`http://localhost:4318`) to export OTLP spans for every request; leave it
unset and tracing stays off. `OTEL_SERVICE_NAME` overrides the default
service name.

The OpenAPI schema is available at [`openapi.yaml`](./openapi.yaml) and, at runtime,
at `http://localhost:8000/openapi.json` / `http://localhost:8000/docs`.

Tear everything down (containers and volumes) with:

```bash
make docker-down
```

## Development without Docker

Requires Python 3.10+ and [uv](https://docs.astral.sh/uv/).

```bash
uv sync --all-extras
# Set DATABASE_URL (Python + FastAPI + SQLite) — see .env.example, then:
uv run alembic upgrade head
uv run uvicorn app.main:app --reload
```

Run `make help` for the full list of common targets.
